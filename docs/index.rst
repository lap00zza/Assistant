Welcome to Assistant's documentation!
=====================================
Assistant is a discord assistant bot. It is a self bot, which means only you 
can use its commands. Assistant runs on top of the Assistant framework.You can 
extend it by writing your own modules/addons or download from a selection of 
awesome community written plugins.

The default plugin comes with these commands, (``modules/utility_commands``):

* ``evaljs``: Evaluate Javascript commands
* ``evalpy``: Evalualte Python commands
* ``sh``: Run shell commands
